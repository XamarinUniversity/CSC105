Before you begin

This is a Xamarin class about C#. You can use either Visual Studio on a Mac or Visual Studio on a Windows. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

In this course, you're going to apply your knowledge from the previous four classes to build a console based journal application. Throughout the course you'll use topics including: classes, methods, properties, and loops.

Objectives

  1. Define a custom type using a class
  2. Create properties to control access to data
  3. Repeat a block of code with a loop
  4. Create methods to add behavior to classes

Lecture

Learning C# with Xamarin: Building a Console Application

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Creating the Journal class
    Exercise
    In this exercise, you will begin to create a console based journal application. You will create a class to represent the journal itself and display a journal message to the console
  2. Creating the JournalEntry class and property
    Exercise
    You are now going to add a JournalEntry class to the application. The JournalEntry class is responsible for storing the text and last edit time of the entry.
  3. Create a console menu
    Exercise
    In this exercise, you will create a console menu system to display possible options to the user
  4. Define and invoke methods
    Exercise
    In this final exercise, you will complete the journal application by creating methods inside the Journal class to lock and unlock.
