Learning C# with Xamarin: Building a Console Application

Duration

15 minutes

Lab goals

In this exercise, you will create a console menu system to display possible options to the user. The menu will be inside of a do while loop so that the user can make multiple selections. To complete the exercise, you will perform the following steps:

  1. Create a do while loop that continues to run until the user quits.
  2. Define the menu options and messages.
  3. Process the user's responses using a switch statement.

Steps

Try to use the picture and above steps to complete the lab exercise, referring to the slides if you need any help in creating the journal application. You can also use the following step-by-step instructions to fill in any knowledge gaps.

Create the menu loop

  1. Inside Program.cs create an int variable to keep track of the user's selections. Place this variable below our code for displaying the journal message.

    Show Code

    int choice;

  2. Create the menu that we will display to the user. You can be creative here and add indents and lines to make the menu more readable. Make sure the menu has options to:
    1. Unlock the journal.
    2. Create a journal entry.
    3. Read a journal entry.
    4. Lock the journal.
    5. Quit the program.

    Show Code

    Console.WriteLine("\nMenu: ");
Console.WriteLine("1) Unlock");
Console.WriteLine("2) Create entry");
Console.WriteLine("3) Read entry");
Console.WriteLine("4) Lock");
Console.WriteLine("5) Quit");
Console.WriteLine("----------------------------------------------------\n");

  3. Run the app to see the output - it should show your menu.
  4. Next, we want the menu to display everytime the user makes a selection. We're going to create a do while loop to manage the menu. Wrap your menu in a do while loop and make sure your loop continues to run as long as the user doesn't select quit.

    Hint: You need to get the users selection before you can create the while condition. Use Console.ReadLine() but remember the value comes back to you as a string!

    Show Code

    do
{
     Console.WriteLine("\nMenu: ");
     Console.WriteLine("1) Unlock");
     Console.WriteLine("2) Create entry");
     Console.WriteLine("3) Read entry");
     Console.WriteLine("4) Lock");
     Console.WriteLine("5) Quit");
     Console.WriteLine("----------------------------------------------------\n");

     choice = int.Parse(Console.ReadLine());
} while(choice != 5);

  5. Run the app and notice that you can keep making selections until you select quit.
