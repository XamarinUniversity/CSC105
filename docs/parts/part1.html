Learning C# with Xamarin: Building a Console Application

Duration

10 minutes

Lab goals

In this exercise, you will begin to create a console based journal application. You will create a class to represent the journal itself and display a journal message to the console. If you use the lab instructions, your output will look something like:

To complete this exercise, you will perform the following steps:

  1. Create a new console application called MyJournal.
  2. Create a Journal class that contains a public string for the message, a private string for the password, and a private bool to idicate if the journal is locked.
  3. Display the journal message into the console.

Steps

Create a new console application

You are going to start by creating a new console application called MyJournal. You may use Visual Studio for Windows, or Visual Studio for Mac to do this.

  1. Launch Visual Studio.
  2. Navigate to File -> New -> Project
  3. Under the Visual C# section, select Console App(.NET Framework) and name the Project MyJournal.
  4. Click OK.

Create the Journal class

You are now going to create the Journal class. The Journal class is responsible for managing the overall state of the journal. For example, you will store the password to unlock the journal here.

  1. Create a new class called Journal. You can create a new class by right clicking on your project and navigating to Add -> Class.
  2. Create the following fields in your class:
    1. A public string to store a startup message called JournalMessage. Initialize this field to a message of your choice.
    2. A private string to store the journal password called journalPassword. Initialize this field to a password of your choice.
    3. A private bool to indicate if the journal is unlocked called journalUnlocked. Initialize this field to false.

    3. Show Code 

    class Journal
{
     public string JournalMessage = "Welcome to my Journal";
     private string journalPassword = "1234";
     private bool journalUnlocked = false;
}

Create an instance of the Journal class

You just created a class to represent a journal, however, a class is only a blueprint. You will now create an actual journal instance (object) by using the new keyword.

  1. Switch to Program.cs.
  2. Inside the Main method create a variable of type Journal called myJournal. You should instantiate this by using the new keyword.
  3. Display the journal's message into the console.

    4. Show Code 

    static void Main(string[] args)
{
     Journal myJournal = new Journal();
     Console.WriteLine(myJournal.JournalMessage);
}

  4. Run the application and verify that the message was displayed.

Summary

In this lab exercise, you created a new console application and began building the structure for the journal class.
